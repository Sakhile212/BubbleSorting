# Sorting Visualizer

## Description
Sorting Visualizer is a web application that demonstrates the Bubble Sort algorithm step by step. Users can generate a random array of integers and watch each comparison and swap visually. The app highlights the elements being compared to help users understand how Bubble Sort works.

## Features
- Generate a random array of integers.
- Visualize Bubble Sort step by step.
- Highlights elements currently being compared.
- Displays all intermediate steps of sorting.
- Responsive design for different screen sizes.

## Technologies Used
- **HTML** – Structure of the application.
- **CSS** – Styling, layout, and responsive design.
- **JavaScript** – Array generation, sorting logic, and DOM manipulation.

## Usage
1. Open `index.html` in a web browser.
2. Click **Generate Array** to create a random array.
3. Click **Sort Array** to visualize Bubble Sort.
4. Watch each step as the algorithm compares and swaps elements.
